# Panchayat
this is a society connect website
## Frontend (Vite + React + Tailwind)
Navigate to the frontend directory:
```bash
cd frontend
# install dependencies
npm install
# start development server
npm run dev
```

## Backend (Node.js + Express + MongoDB)
Navigate to the backend directory:
```bash
cd backend
# install dependencies
npm install
# start the server
npm start
```
**Note:** Ensure a MongoDB instance is running and the connection string in `backend/server.js` points to your database.

🏘️ Panchayat
Panchayat is a residential society management platform designed to bridge the gap between neighbors, foster community engagement, and simplify access to trusted local services. By digitizing the "community life" experience, Panchayat aims to turn residential complexes into genuine support networks.

🌟 Core Pillars
The Trust Circle: A vetted directory of reliable house help, tradespeople, and service providers recommended by neighbors you actually trust.

The Chaupal: A central community hub for digital discussions, event planning, and local updates.

The Vidhan: A structured space for society governance, rule-sharing, and community decision-making.

🛠️ Technical Implementation
Panchayat is being developed as a robust, scalable platform using the MERN Stack:

MongoDB: For flexible, schema-based storage of resident directories and community data.

Express & Node.js: Powering the backend API with a focus on clean, modular architecture.

React: Providing a dynamic, responsive frontend that prioritizes ease-of-use for all age groups.

PWA Readiness: Currently being architected to support offline access and mobile-first functionality, ensuring residents can access essential information without relying on constant connectivity.

🏗️ Architectural Vision
The project is built on Clean Architecture principles to ensure that the logic behind "The Trust Circle" and "The Vidhan" remains decoupled from the UI, allowing for future scalability—whether that means adding new modules or integrating IoT-based society security features.
